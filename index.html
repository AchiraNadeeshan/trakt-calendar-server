Generate an iCal calendar from your Trakt.tv shows and movies. This allows you to subscribe to your Trakt activity in calendar apps like Google Calendar, Apple Calendar, or Outlook.

API Usage (v1)

Send a GET request to the following endpoint:

/api/v1/ical?token=YOUR_TRAKT_TOKEN&days=30

Query parameters:

  • token (required): Your Trakt personal access token
  • days (optional): Number of days to include (default: 30)

Generate Your Calendar URL

https://trakt-calendar-server.vercel.app/api/v1/ical

Use this URL to subscribe in your calendar app.
